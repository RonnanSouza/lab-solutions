# Lab solutions

Lab solutions is a software to help you manage your clinic lab.

## Setup

`make setup`

## Tests

`make test`

## Running backend

- `make deps` to start the database
- `make back`

~~~
This will also mock some entries to be easier to test
~~~

## Running frontend
